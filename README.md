# Finansal-Zaman-Serileri-Analizi
Statistical analysis of financial assets and portfolio optimization using Python, featuring Block Maxima extreme value theory.
# Finansal Varlıkların İstatistiksel Analizi ve Portföy Yönetimi

Bu proje, seçili finansal varlıkların fiyat ve getiri hareketlerini Python kullanarak analiz etmeyi amaçlamaktadır. Çalışma kapsamında temel istatistikler, zaman serisi analizleri ve uç değer teorisi (Blok Maksima) uygulanmıştır.

## Proje İçeriği
* **Veri Görselleştirme:** Kapanış grafikleri, histogramlar ve boxplot (kutu) grafikleri ile veri dağılımının incelenmesi.
* **Uç Değer Dağılımları:** Blok Maksima yöntemi ile finansal şoklara karşı risk analizi.
* **Portföy Analizi:** Tekil hisse ile portföy performansının karşılaştırılması ve hisseler arası korelasyon incelemesi.

## Kullanılan Teknolojiler
* Python (Pandas, Matplotlib/Seaborn, Scipy)
